# Particle System

Drive many small elements with simple per-particle rules to get emergent, organic motion — 2D canvas for hundreds, GPU `Points` for thousands. For web and creative developers building ambient backgrounds, celebratory bursts, weather, flow fields, and connected-dot networks that stay performant at scale.

## When it activates

- "Build a particle system" or "make confetti / snow / smoke / sparks."
- "Create a connected-dot / constellation network background."
- "Add a flow-field or curl-noise particle effect."
- "Render thousands of GPU particles with Three.js Points" or "animate emitters with forces."

## Example prompts

- "Build a constellation background that links nearby particles and stays O(n) past 2000 dots."
- "Make a curl-noise flow field that swirls particles like fluid and evolves over time."
- "Create a confetti burst that reads as confetti — rotating rects, gravity, and air drag."

## What's inside

- `SKILL.md` — the per-particle integration loop, composable forces, flow fields and curl noise, burst vs continuous emission, the spatial-grid connected-dot technique, and a GPU `Points` + shader system.
- `references/particle-recipes.md` — complete canvas confetti/snow/smoke systems, mouse attraction/repulsion, full simplex flow-field and curl-noise fields, the end-to-end connected-dot background, and a GPU `Points` system with per-particle life/seed attributes.
